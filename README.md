# APIContext (apicontext)
APIContext (formerly APImetrics) is an advanced synthetic API testing and monitoring platform that measures API performance, enforces SLOs, and validates API conformance for critical APIs. It provides an API directory with performance data on 300+ top API providers and offers solutions for developers, enterprises, and API partners.

## Features

| Name | Description |
|------|-------------|
| Synthetic API Testing | Continuously test APIs from multiple global locations to measure latency, availability, and correctness. |
| SLO Monitoring | Define and enforce Service Level Objectives for API performance with threshold-based alerting. |
| API Conformance Testing | Validate API responses against expected schemas and behavioral contracts to detect regressions. |
| API Directory | Access performance data and metrics on 300+ top API providers for benchmarking and comparison. |
| Performance Dashboards | Real-time and historical dashboards showing API performance trends, error rates, and SLO compliance. |
| Multi-Location Testing | Test APIs from multiple geographic locations to identify regional performance variations. |

## Use Cases

| Name | Description |
|------|-------------|
| API Performance Monitoring | Continuously monitor critical API endpoints for latency, uptime, and performance degradations. |
| SLO Compliance Reporting | Track API performance against defined SLOs and generate compliance reports for stakeholders. |
| API Regression Detection | Detect API breaking changes and response schema violations using continuous conformance testing. |
| Third-Party API Benchmarking | Compare API provider performance using the APIContext directory of 300+ top providers. |
| Partner API Oversight | Monitor partner API performance and SLA compliance from a developer perspective. |
